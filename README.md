# 介绍
此项目是一个拥有基本功能的使用对象存储的局域网网盘, 目前支持使用腾讯云的对象存储和本地存储, 网盘的主体使用springboot和mysql完成(有点想换成h2),
最终目的是实现脚本文件下载的整合和挂机省电下载的，如脚本下载完图片就通过接口上传到网盘，强迫症发作要查看不在身边的文件的内容，百度云挂机下载(虽然不一定做得到)
***
