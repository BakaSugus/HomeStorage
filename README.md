# 介绍
这个项目主要完成网盘的基本功能
***

# 功能
- [x] 上传，下载文件
- [x] 多用户
- [x] 投屏 需搭配插件使用 
- [x] 基本文件预览
- [x] 压缩下载 批量下载
- [x] 分片上传
- [x] 文件夹共享
- [x] 重命名
- [x] 批量下载
- [x] http下载
- [ ] 播放历史
- [ ] office文件预览
- [ ] torrent下载
- [ ] 共享资源的编辑和版本控制的协调
